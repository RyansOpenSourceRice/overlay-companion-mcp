Add RDP/VNC/KasmVNC connection management to MCP server

Implements programmatic connection configuration for AI agents while
preserving manual web interface configuration for end users.

Features:
- ConnectionManagementService: Persistent storage, validation, testing
- 5 new MCP tools: add/list/test/remove/set_active_connection
- Protocol-specific validation (RDP requires username+password)
- KasmVNC recommended for multi-monitor support
- TCP connectivity testing before use
- Thread-safe operations with semaphore locking

Documentation:
- SPECIFICATION.md: Protocol comparison and tool specifications
- docs/CONNECTION_MANAGEMENT.md: Usage guide with examples
- INTEGRATION_TODO.md: Integration checklist
- Web interface: AI integration notice
